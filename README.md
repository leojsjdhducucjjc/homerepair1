# New Website Project

Contractor-style landing page with a Node + Express server so you can run it with npm.

## Project structure

- `index.html`
- `src/css/styles.css`
- `src/js/main.js`
- `assets/`

## Run with npm

```bash
cd new-website-project
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Production run

```bash
npm start
```
